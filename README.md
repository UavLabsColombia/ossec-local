## Scripts para ossec

### Rápida instalación 

```
curl -k https://raw.githubusercontent.com/dorancemc/ossec-local/master/ossec_install.sh | sh -x
```

### Para instalar el server web en CentOS/RHEL
```
yum install ossec-wui
```
